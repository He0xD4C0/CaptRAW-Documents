## 🗓️ 详细开发计划

### **第1阶段：前端开发（5-7天）**

#### **第1天：项目初始化与基础架构**
1. **创建项目结构**
   ```bash
   mkdir CaptRAW-Documents
   cd CaptRAW-Documents
   mkdir frontend backend shared docs
   ```

2. **前端项目初始化**
   ```bash
   cd frontend
   npx create-react-app . --template typescript
   ```

3. **安装核心依赖**
   ```bash
   npm install react-router-dom axios react-query @tanstack/react-query
   npm install tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   ```

4. **配置Tailwind CSS**
   - 更新 `tailwind.config.js`
   - 配置 `index.css` 引入Tailwind
   - 设置基础颜色主题（深色/浅色）

5. **创建项目结构**
   ```
   frontend/src/
   ├── components/
   │   ├── layout/
   │   ├── ui/
   │   └── shared/
   ├── pages/
   ├── hooks/
   ├── services/
   ├── utils/
   ├── types/
   └── assets/
   ```

### **第2阶段：数据库与后端开发（6-8天）**

#### **第8天：数据库设计**
1. **初始化后端项目**
   ```bash
   cd ../backend
   npm init -y
   npm install express typescript ts-node @types/node @types/express
   npm install prisma @prisma/client
   npm install bcryptjs jsonwebtoken cors dotenv
   ```

2. **创建Prisma Schema** (`prisma/schema.prisma`)
   - User模型（用户）
   - Article模型（文章）
   - Comment模型（评论）
   - Tag模型（标签）
   - Notice模型（公告）
   - Banner模型（横幅）

3. **配置数据库连接**
   - 设置环境变量
   - 配置PostgreSQL连接

4. **生成Prisma客户端**
   ```bash
   npx prisma generate
   npx prisma migrate dev --name init
   ```

#### **第9天：用户认证系统**
1. **创建Express应用结构**
   ```
   backend/
   ├── src/
   │   ├── controllers/
   │   ├── services/
   │   ├── middleware/
   │   ├── utils/
   │   └── app.ts
   ```

2. **实现用户注册API** (`POST /api/auth/register`)
   - 输入验证
   - 密码加密
   - 邮箱唯一性检查

3. **实现用户登录API** (`POST /api/auth/login`)
   - 密码验证
   - JWT令牌生成

4. **实现认证中间件**
   - JWT验证
   - 角色权限检查

#### **第10天：文章管理API**
1. **实现文章CRUD API**
   - `GET /api/articles` - 获取文章列表（分页、筛选）
   - `GET /api/articles/:id` - 获取单篇文章
   - `POST /api/articles` - 创建文章（需要认证）
   - `PUT /api/articles/:id` - 更新文章（需要权限）
   - `DELETE /api/articles/:id` - 删除文章（需要权限）

2. **实现文章搜索API**
   - 全文搜索
   - 标签搜索
   - 作者搜索

3. **实现文章统计API**
   - 阅读量统计
   - 点赞功能

#### **第11天：公告与横幅API**
1. **实现公告管理API**
   - `GET /api/notices` - 获取公告列表
   - 公告CRUD操作（管理员权限）

2. **实现横幅管理API**
   - `GET /api/banners` - 获取横幅列表
   - 横幅CRUD操作（管理员权限）

3. **实现标签管理API**
   - `GET /api/tags` - 获取标签列表
   - 标签云统计

#### **第12天：评论与用户API**
1. **实现评论系统API**
   - `GET /api/articles/:id/comments` - 获取评论
   - `POST /api/articles/:id/comments` - 添加评论
   - 评论审核功能

2. **实现用户管理API**
   - `GET /api/users/:id` - 获取用户信息
   - `PUT /api/users/:id` - 更新用户信息
   - 用户头像上传

3. **实现管理员API**
   - 用户管理
   - 内容审核
   - 系统统计

#### **第13天：文件上传与优化**
1. **实现文件上传API**
   - 图片上传
   - 文件大小限制
   - 文件类型检查

2. **实现API文档**
   - Swagger/OpenAPI配置
   - API测试端点

3. **实现缓存机制**
   - Redis缓存配置
   - 文章缓存策略

4. **实现速率限制**
   - API调用限制
   - 防止滥用

#### **第14天：前后端集成**
1. **配置CORS**
2. **创建API服务层** (frontend/src/services/)
   - 用户服务
   - 文章服务
   - 公告服务

3. **集成React Query**
   - 数据获取
   - 缓存管理
   - 错误处理

4. **实现表单提交**
   - 文章投稿
   - 评论提交
