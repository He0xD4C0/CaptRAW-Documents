## 🔧 可用工具和脚本

### 前端工具:
- `npm start` - 启动开发服务器
- `npm run build` - 生产构建
- `npm test` - 运行测试
- 配置同步脚本自动执行

### 后端工具:
- `npm run dev` - 开发服务器
- `npm run build` - TypeScript编译
- `npm start` - 生产服务器
- `npm run upload-assets` - 资产上传脚本
- `npm run test-db` - 数据库健康检查

### 全局脚本:
- `scripts/init-minio-local.sh` - MinIO本地初始化
- `scripts/init-minio.sh` - MinIO生产初始化
- `docker-compose.yml` - 容器化环境
