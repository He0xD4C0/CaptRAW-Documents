## 🔧 技术决策要点

### 1. 数据库选择
- 使用 **PostgreSQL** 作为主数据库
- 使用 **Prisma ORM** 进行数据访问（后续集成）
- 当前阶段使用原始SQL以保持简单

### 2. 存储协议
- 使用 **S3协议** 作为标准接口
- 支持 **MinIO, AWS S3, 阿里云OSS** 等通用对象存储兼容实现
- 通过配置切换不同存储后端

### 3. 资产标识
- 保持现有的 **assetUuid** 系统
- 数据库存储真实的对象键映射
- 前端不感知实际存储路径

### 4. 安全性
- 签名URL有效期设为 **5分钟**
- 私有资产需要 **用户认证**
- 配置信息 **隔离敏感字段**
