## 📋 项目结构现状

```
frontend/src/
├── data/                    # 数据文件
│   ├── articles.json       # 文章数据
│   ├── notices.json        # 公告数据
│   └── banners.json        # 横幅数据
├── types/                  # 类型定义
│   └── index.ts           # 所有类型定义
├── services/              # API服务
│   └── api.ts            # 主API服务
├── hooks/                 # React Query Hooks
│   ├── useArticles.ts    # 文章Hooks
│   ├── useNotices.ts     # 公告Hooks
│   ├── useBanners.ts     # 横幅Hooks
│   └── useTags.ts        # 标签Hooks
├── components/           # 组件
│   ├── layout/          # 布局组件
│   ├── ui/              # UI组件
│   ├── home/            # 主页组件
│   └── articles/        # 文章组件
└── pages/               # 页面组件
```
