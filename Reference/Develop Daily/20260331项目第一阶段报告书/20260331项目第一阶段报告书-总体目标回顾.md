## 🎯 项目总体目标回顾

### 原始项目总览
- **项目名称**: CaptRAW Documents  
- **网站名称**: CaptRAW Documents  
- **核心目标**: 创建一个完整的博客系统，包含用户管理、文章管理、文章浏览功能  
- **开发顺序**: 前端 → 数据库/后端 → 文档完善  
- **预计总时间**: 15-20天（持续工作）  
- **当前阶段**: 第一阶段 - 前端开发（计划5-7天）

### 第一阶段原始计划（5-7天）
根据《20260330项目企划书.plan.md》，第一阶段详细计划如下：

#### 第1天：项目初始化与基础架构
- 创建项目结构
- 前端项目初始化（Create React App + TypeScript）
- 安装核心依赖（React Router, Axios, React Query, Tailwind CSS）
- 配置Tailwind CSS
- 创建基础项目结构

#### 第2天：布局组件开发
- 创建Layout组件
- 创建Header组件（基于伪代码实现）
- 创建Footer组件
- 创建主题切换功能

#### 第3天：主页组件开发
- 创建HomePage组件
- 创建BannerCarousel组件（轮播功能）
- 创建NoticeBoard组件（公告板）
- 创建ArticleTimeline组件（文章时间线）

#### 第4天：文章相关页面
- 创建ArticlePage组件（文章详情）
- 创建ArticleCard组件（文章卡片）
- 创建MarkdownRenderer组件（Markdown解析）
- 创建CommentSection组件（评论系统）

#### 第5天：用户认证页面
- 创建LoginPage组件（登录）
- 创建RegisterPage组件（注册）
- 创建ProfilePage组件（个人资料）
- 创建SettingsPage组件（设置）

#### 第6天：管理页面
- 创建AdminDashboard组件（管理仪表板）
- 创建ArticleManagement组件（文章管理）
- 创建NoticeManagement组件（公告管理）
- 创建BannerManagement组件（横幅管理）

#### 第7天：投稿与搜索功能
- 创建SubmitArticlePage组件（投稿）
- 创建DirectoryPage组件（目录）
- 创建SearchPage组件（搜索）
- 创建404页面和加载状态
