## 📋 使用说明

### 数据库管理
```bash
# 重置数据库（删除并重新创建）
cd backend && npm run reset-db

# 填充示例数据
cd backend && npm run seed-db

# 上传资产到MinIO
cd backend && npm run upload-assets
```

### API使用示例
```bash
# 获取运行时配置（前端使用）
curl http://localhost:3001/api/server-info/runtime-config

# 验证配置完整性
curl http://localhost:3001/api/server-info/validate

# 按分类获取配置
curl http://localhost:3001/api/server-info/category/general
```

### 添加新配置
1. 在`seed-data.sql`中添加新的INSERT语句
2. 或通过API动态添加：
```bash
curl -X POST http://localhost:3001/api/server-info/admin/set \
  -H "Content-Type: application/json" \
  -H "x-admin: true" \
  -d '{
    "info_key": "new_config_key",
    "info_value": "config_value",
    "description": "配置描述",
    "is_public": true,
    "category": "general"
  }'
```
